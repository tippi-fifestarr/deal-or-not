# Deal or NOT — Technical Review

*Reviewed 2026-03-08 at 5:55 PM*

## 5-Minute Judge Overview

Deal or NOT is a fully on-chain "Deal or No Deal" game that uses 5 Chainlink products working together. The team iterated through 4 architectural approaches (documented in Whitepaper.md) before landing on CRE as the correct solution for provably fair, private case values.

## Chainlink Integration (5 Products)

| Product | Purpose | Where |
|---------|---------|-------|
| **VRF** | Fair shuffle seed for case assignment | `DealOrNotQuickPlay.sol` / `DealOrNotConfidential.sol` |
| **CRE** | Confidential case reveal (entropy + value computation in enclave) | `workflows/confidential-reveal/main.ts` |
| **CRE Confidential HTTP** | AI Banker via Gemini 2.5 Flash (private game state never exposed) | `workflows/banker-ai/main.ts` + `gemini.ts` |
| **Price Feeds** | ETH/USD conversion for payouts | `SharedPriceFeed.sol` / `PriceFeedHelper.sol` |
| **CCIP** | Cross-chain play (ETH Sepolia to Base Sepolia) | `DealOrNotBridge.sol` + `DealOrNotGateway.sol` |

## Architecture

```
Player (Browser) -> Smart Contract (Base Sepolia) -> CRE Workflow (enclave)
                                                   -> VRF (shuffle)
                                                   -> Gemini (banker AI)
                                                   -> Price Feed (ETH/USD)
                                                   -> CCIP (cross-chain)
```

The key insight: case values must be hidden from everyone (including the node operator) until opened. CRE Confidential HTTP solves this by computing values inside a TEE enclave using external entropy, so no party can predict or manipulate outcomes.

## What Makes This Impressive

1. **Real iteration** — Team tried commit-reveal, ZK proofs, Chainlink Functions, then CRE. Documented why each failed and why CRE is the right answer.
2. **CRE as the killer feature** — Confidential HTTP fetches external entropy inside a TEE, computes case values privately, and writes results on-chain. No party can cheat.
3. **AI Banker personality** — Gemini generates snarky, context-aware messages referencing actual game state. Runs through CRE so game data stays private.
4. **Production-grade** — Verified contracts, 47 tests, cross-chain bridge, sponsor jackpot system, agent infrastructure for AI players.
5. **Live and playable** — Not a prototype. Real games played with real wallets on Base Sepolia.
